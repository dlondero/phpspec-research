This is a repo used to play with phpspec and compare the same tests with PHPUnit.

### Install dependencies
```bash
$ composer install
```

### Run phpspec
```bash
$ bin/phpspec run
```

### Run PHPUnit
```bash
$ bin/phpunit --colors tests/MarkdownTest.php
```

```bash
$ bin/phpunit --testdox tests/MarkdownTest.php
```
